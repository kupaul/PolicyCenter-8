RICededPremiumTransaction (Virtual) (delegates to Extractable, SimpleEffDated)
Description
A delegate for RICededPremiumTransaction, which is an individual ceding calculation done between a direct premium Transaction which insures an RIRisk, and a RIAgreement which reinsures that RIRisk.
Entity: RICededPremiumTransaction
Attributes: Extendable?, Final?
