BasedOnUntyped  Derived property returning EffDated     (virtual property)?
Return the bean from which this bean was based (cloned), or null if this bean is new to this branch. This value will never change for the life of the bean as its set when the bean is first created, either as a bean new to this branch (in which case it's null) or when this bean is cloned from a bean in an existing branch when its branch is created.

When compiling DiffItems a bean is always compared to it's based-on bean to discover what fields on it have changed. If there is no based-on bean then we know that this bean was new to this branch.
