AnswerContainer (Virtual)
Description
A delegate for entities that hold answers to questions.
Entity: AnswerContainer
Attributes: Extendable?, Final?
Fields

AssociatedPolicyPeriod  Derived property returning PolicyPeriod     (virtual property)?
Returns policy period associated with this AnswerContainer. If there is no associated policy period, override this function to return null. In this case, the check to see if QuestionSets are disabled in offerings will be ignored for all QuestionSets on this AnswerContainer.
