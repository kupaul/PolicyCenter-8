AvailableProductOffers  Derived property returning java.util.List     (virtual property)?
Returns the list of all *available* ProductOffers. This is almost the same as calling getProductOffers, except:

  • An ProductOffer with status = unavailable won't be returned
  • If OtherProdsAvail = true, then all products not in the results will be returned as an available ProductOffer.
