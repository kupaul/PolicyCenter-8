FormEdgeTable (pc_formedgetable) (delegates to EffDated)
Description
This entity exists because we don't currently support edge foreign keys between effdated entities. This edge table is used by the SupersededForms array on Form to link a new Form entity to the old Form that it is replacing. An edge table is necessary to avoid circular references in the datamodel.
