Product Question Sets Page

Manage the question sets attached to a product on the product Question Sets page.

This page lists all of the question sets associated with the selected product. Click a question set name to open that question set’s home page. On the question set home page, you also can access links to configure the individual questions, question set availability, and offerings pertaining to the question set.

You also can access the individual question sets until Question Sets in the navigation panel.

Adding a Question Set

To add an existing question set to the selected product

1. Click Add.

2. From the drop-down menu that appears, select the question set to add to the selected product.

Note:  In the Product Question Sets page, you can only add existing question sets to the selected product. You cannot define new question sets. You define new question sets on the main Question Sets page.

Common Operations

To revert changes to product question sets

The changes you make in the current page are stored in your active change list. You can

• Cancel all changes you have made in the current page by clicking Revert.

• Selectively cancel changes by clicking Changes to display the Changes panel, then clicking to open the Changes page, then clicking Revert in the Revert column.

Note:  When you click Revert, Product Designer reloads the entire product model from the workspace.

 

To remove question sets from the selected product

Removing a question set from a product does not delete the question set from PolicyCenter. It only removes it from the selected product.

1. Select the question set to remove by clicking anywhere in the row except the blue question set name, which is a link. To delete multiple items at once, select multiple rows.

2. Click Remove.

Removed question sets are added to your active change list. They are not removed from the PolicyCenter product configuration until you commit your changes.

To navigate to related pages

Use the links under Go to to display pages where you can specify additional product properties.
