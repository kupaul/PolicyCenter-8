Question Set Questions Page

Define the individual questions that belong to this question set. You can add, delete, edit, and rearrange questions.

Adding a New Question

1. Click one of the five question types at the top of the page to display the Add Question dialog box for the corresponding question type.

 

String question type. Set the Format to:

• String Field – Accepts a single line of text.

• String Text Box – Accepts multiples lines of text. Displays a scroll bar when needed.

Integer question type. Set the Format to:

• Integer Field – Accepts a single line of integers.

Boolean question type. Set the Format to:

• Boolean Checkbox – Displays a check box that can be selected for Yes or cleared for No.

• Boolean Radio – Displays two option buttons labeled Yes and No.

• Boolean Select – Displays a drop-down list with two selections labeled Yes and No.

Note: The actual labels that appear in PolicyCenter are determined by PolicyCenter configuration. A developer uses studio to configure these labels.

Choice question type. Set the Format to:

• Choice Radio – Displays a vertically-oriented set of option buttons, one for each choice you add in the Choices page.

• Choice Select – Displays a drop-down list of selections, one for each choice you add in the Choices page.

Date question type. Set the Format to:

• Date Field – Displays a date field that accepts a manually entered date or a date inserted with an adjacent Calendar button.

Note: The date format that appears in PolicyCenter is determined by the regional settings selected by each PolicyCenter user.

The new question is added at the end of the list and the Question Editor appears to the right of the question. If needed, scroll down to see the new question, and enlarge your browser window or scroll to the right to see the editor panel.

2. Specify the details of the question in the Question Editor, according to the question type:

 

Code

Unique identifier for the question, specified when you added the question. Established when you add a new question and thereafter cannot be changed.

Label

Text of the question, initially specified when you added the question set. You can:

 

• Edit the text directly in the field using the current language. You can set the current language in the User Settings page.

• Click Translate to display the Display Key Values by Language dialog box, where you can enter or edit the text in any supported language.

 

 

Indent (characters)

Number of characters to indent the question label from the left margin. Enables you to create questions that appear subordinate to other questions according to their level of indent.

Required

Whether the question must be answered before continuing to the next wizard step in PolicyCenter.

Post

Whether answering this questions triggers a post-on-change within PolicyCenter. The post-on-change operation causes PolicyCenter to reload the page with new values. One of:

• Automatic – PolicyCenter reloads the page if visibility of a follow-up question is dependent on the answer to this question.

• Always – PolicyCenter always reloads the page when this question is answered. Use this option when a non-question field on the page is dependent on the answer to this question.

Help Text

Helpful instructions or information pertaining to answering the question. Click Add to add a new row of help text. Regardless of how the text wraps in the entry field, it appears on a single line in PolicyCenter. To insert more lines of help text, click Add to add more rows. For each row of help text, you can:

 

• Edit the text directly in the field using the current language. You can set the current language in the User Settings page.

• Click Translate to display the Display Key Values by Language dialog box, where you can enter or edit the text in any supported language.

 

 

 

• Click Delete to delete the help text row from the question.

Dependent On

Displays the Dependent On page for this question where you can select one or more questions whose answers determine whether the current question is visible in PolicyCenter.

Incorrect Answer

Displays the Incorrect Answer page for this question where you can define the needed behavior of incorrect answers.

Availability

Displays the Availability page where you can define the availability of this question.

Choices

Appears only for choice question types.

Displays the Choices page where you can define each of the choices for this question.

3. If needed, specify a default answer to for each question by selecting or filling in the answer on the Questions page.

4. Repeat steps 1 and 2 to define the entire question set.

Editing Questions

1. In the Questions page, click the question you want to edit. The Question Editor appears to the right of the question. If needed, enlarge your browser window or scroll to the right to see the editor panel.

2. Use the information in the preceding table to edit the question properties as needed.

3. Use the links at the bottom of the Question Editor to open pages where you can configure details such as Dependent On, Incorrect Answer, Availability, and Choices.

Note:  You cannot change the question Code, Format, or Type of an existing question. You must add a new question to specify these properties.

Alternatively, select a question in the Navigation panel to edit individually it in the Question home page.

Rearranging Questions

The order of questions in the Questions page determines the order in which they appear in PolicyCenter.

To rearrange questions

1. Click a question to rearrange, then drag it to a new position in the list of questions.

2. Repeat with other questions until all questions are ordered as needed.

Resetting Questions

You can specify a default answer for each question by selecting or filling it in on the Questions page.

To reset the default answer for a question

• Click a question to reset, then click Reset To Default.

Deleting Questions

1. In the Questions page, click the question you want to delete. The Question Editor appears to the right of the question. If needed, enlarge your browser window or scroll to the right to see the editor panel.

2. Click Delete to delete the question.

The deleted item is added to your active change list. It is not deleted from the PolicyCenter configuration until you commit your changes.

Prior to committing your change list, you can click Cancel Deletion to restore the deleted item. The gray overlay is removed and you can once again edit the item.

 

 

 

Common Operations

To revert changes to the question set

 

The changes you make in the current page are stored in your active change list. You can

• Cancel all changes you have made in the current page by clicking Revert.

• Selectively cancel changes by clicking Changes to display the Changes panel, then clicking to open the Changes page, then clicking Revert in the Revert column.

Note:  When you click Revert, Product Designer reloads the entire product model from the workspace.

 

 

 

 

To validate changes to the question set

Validate your changes by clicking Validate. Validation errors appear at the top of the page on a red background. The errors are marked with and appear on a red background. When no validation errors exist, clicking Validate displays the message No validation errors marked with on a green background.

Note:  When you click Validate, Product Designer reloads the entire product model from the workspace. Therefore, Product Designer can display unexpected values after validation if other users have committed changes to the same workspace.

 

 

 

To navigate to related pages

Use the links under Go to to display pages where you can specify additional question set properties.
