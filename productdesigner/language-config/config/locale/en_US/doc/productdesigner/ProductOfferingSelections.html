Product Offering Selections Page

Examine and manage offering selections. Offering selections are the attributes of the product that can be enabled or disabled as a part of the offering. Offering selections appear as a list that can be expanded to locate the selections you want to enable or disable. The selections that can be individually enabled or disabled as part of an offering include:

• Policy lines, in package policies only

• Policy terms

• Coverages

• Coverage terms

• Cover term options and packages

• Exclusions

• Conditions

• Modifiers

• Question sets

• Individual questions from question sets

The Selections table contains the following columns:

 

Column

Description

Name

Selection to include or exclude from the offering:

Expands the selection and reveals its descendant selections.

Collapses the selection and hides its descendant selections.

Indicates a selection with no descendants.

Enabled

Indicates whether the attribute is enabled in this offering. Displays if the attribute is enabled. The entry is blank if the attribute is not enabled.

Modified

Indicates whether the attribute or one of its descendants is modified in the selected offering:

(blank) Attribute is not modified in this offering.

Attribute is modified in this offering.

Attribute is not modified in this offering, but one or more if its descendants is modified. Expand descendant nodes to locate the modified attributes.

Attribute and one or more of its descendants are modified in this offering. Expand descendant nodes to locate the modified attributes.

 

 

Other operations

• Sort – Click the Name column header to sort by that column. Click again to reverse the sort order.

• Show/hide columns – Point to any column header to display a selector at the edge of the column header. Click the selector then click Columns, and then select or clear check boxes for the columns you want to show or hide, respectively.

 

Modifying Offering Selections

1. Click the row that represents the selection you want to modify. If the selection can be modified, the Selection Editor appears to the right of the selection. If you cannot see the Selection Editor, enlarge your browser window or scroll to the ri ght.

2. Make the needed changes to the selection. The Selection Editor displays the value of each item in the selection as it exists in the base configuration of the policy. The items that can be changed vary depending on the selection. Many items can only be enabled or disabled. Others can have different existence per offering. Still others can have different values or different minimum and maximum values per offering.

3. Repeat steps 1 and 2 to fully define the offering. All changes are saved as you make them, so there is no save command.
