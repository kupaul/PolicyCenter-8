Editing Basic Policy Line Modifier Rate Factor Settings

 

Rate Factor Type

Rate factor type specified when the rate factor was added. Select a different type from this list if needed. Rate factor types are defined by configuring the PolicyCenter data model, as explained in the Product Model Guide.

 

Default Minimum

Default minimum value for this rate factor, if not overridden by a specified state minimum value. This value is also limited by the overall default minimum value specified in the Policy Line Modifier home page.

Default Maximum

Default maximum value for this rate factor, if not overridden by a specified state maximum value. This value is also limited by the overall default maximum value specified in the Policy Line Modifier home page.

Common Operations

To cancel addition or revert changes to policy line modifier rate factor settings

 

The changes you make in the current page are stored in your active change list. If you have just added an item, click Cancel Addition to remove the newly-added item. If you are editing an existing item, cancel all changes you have made in the current page by clicking Revert. Or, selectively cancel changes by clicking Changes to display the Changes panel, then click Show Full Panel to open the Changes page.

 

 

 

 

To validate the policy line modifier rate factor

Validate your changes by clicking Validate. Validation errors appear at the top of the page on a red background. The errors are marked with and appear on a red background. When no validation errors exist, clicking Validate displays the message No validation errors marked with on a green background.

Note:  When you click Validate, Product Designer reloads the entire product model from the workspace. Therefore, Product Designer can display unexpected values after validation if other users have committed changes to the same workspace.

 

 

To delete the policy line modifier rate factor

While editing the policy line modifier rate factor settings in the Policy Line Modifier Rate Factor home page, click Delete.

• If you have not yet committed the policy line modifier rate factor, Product Designer deletes the rate factor.

• If the policy line modifier rate factor has been committed, the portion of the page that was previously editable becomes gray and you cannot make further changes to it.

The deleted item is added to your active change list. It is not deleted from the PolicyCenter configuration until you commit your changes.

Prior to committing your change list, you can click Cancel Deletion to restore the deleted item. The gray overlay is removed and you can once again edit the item.

 

 

To navigate to related pages

Use the links under Go to to display pages where you can specify additional policy line modifier rate factor properties.
