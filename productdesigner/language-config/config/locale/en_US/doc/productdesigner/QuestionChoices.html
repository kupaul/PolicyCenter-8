Question Choices Page

Configure choices for question of type Choice. You must define at least one choice for each choice-type question.

 

Column

Description

Sequence

Order in which the choice appears in PolicyCenter, relative to other choices for this question.

Code

Unique identifier for the question choice. Must begin with a letter and be composed only of letters, numbers, and underscores.

Name

Choice name that appears in the drop-down list in PolicyCenter. You can:

 

• Edit the text directly in the field using the current language. You can set the current language in the User Settings page.

• Click Translate to display the Display Key Values by Language dialog box, where you can enter or edit the text in any supported language.

 

 

Description

Choice description. You can:

 

• Edit the text directly in the field using the current language. You can set the current language in the User Settings page.

• Click Translate to display the Display Key Values by Language dialog box, where you can enter or edit the text in any supported language.

 

 

Score

Unused in the default PolicyCenter configuration. You can configure this field by using Studio to write Gosu code that returns the score value according to the selected answer. Using this numeric value enables more flexibility than a simple correct/incorrect answer decision.

 

Other operations

• Sort – Click any column header to sort by that column. Click again to reverse the sort order.

• Change order – If the table is sorted by Sequence, drag a single row to change the order in which it is listed. Alternatively, edit the sequence number to establish a new order. Sequence determines the order in which items appear in PolicyCenter PCF objects when they are sorted by Priority.

• Edit – Click the blue item name to open a page where you can edit the item.

 

 

 

Adding a New Choice

1. Click Add to add a new row to the Choices table. Note that newly-added choices are is added to the end of the list. If needed, scroll down to see the new row.

2. Fill in the fields in the Choices table to define the choice.

3. Repeat steps 1 and 2 to add more choices to the selected question.

Common Operations

To revert changes to question choices

The changes you make in the current page are stored in your active change list. You can

• Cancel all changes you have made in the current page by clicking Revert.

• Selectively cancel changes by clicking Changes to display the Changes panel, then clicking to open the Changes page, then clicking Revert in the Revert column.

Note:  When you click Revert, Product Designer reloads the entire product model from the workspace.

 

 

 

To validate question choices

Validate your changes by clicking Validate. Validation errors appear at the top of the page on a red background. The errors are marked with and appear on a red background. When no validation errors exist, clicking Validate displays the message No validation errors marked with on a green background.

Note:  When you click Validate, Product Designer reloads the entire product model from the workspace. Therefore, Product Designer can display unexpected values after validation if other users have committed changes to the same workspace.

 

 

To navigate to related pages

Use the links under Go to to display pages where you can specify additional question properties. The available links vary depending on the question type.
