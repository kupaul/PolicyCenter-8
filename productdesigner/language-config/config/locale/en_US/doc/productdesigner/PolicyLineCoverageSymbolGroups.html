Policy Line Coverage Symbol Groups Page

In the base configuration, coverage symbol groups apply only to the commercial auto line in North America. PolicyCenter prints coverage symbols on the policy declarations page to indicate the types of coverages in effect.

To use coverage symbols, you must:

1. Use Studio to:

• Add the Coverage Symbol Groups tab and field to a policy line pattern.

• Define appropriate coverage symbols in the CoverageSymbolType typelist.

• Write Gosu code to initialize the coverage symbol groups, tie them to specific coverages, and set or clear the coverage group check boxes to correspond to selected coverages.

For more information, see the Product Model Guide.

2. Use Product Designer to:

• Define new coverage symbol groups, if needed.

• Assign coverage symbol groups to coverage symbol patterns.

The table on the Coverage Symbol Groups page lists the coverage groups that have already been defined.

 

Column

Description

Sequence

Order in which the coverage group appears in PolicyCenter, relative to other coverage groups.

Name

Name of the coverage group. Click the name to open the Coverage Symbol Group home page where you can assign the coverage group to one or more coverage symbol patterns.

Other operations

• Sort – Click any column header to sort by that column. Click again to reverse the sort order.

• Change order – If the table is sorted by Sequence, drag a single row to change the order in which it is listed. Alternatively, edit the sequence number to establish a new order. Sequence determines the order in which items appear in PolicyCenter PCF objects when they are sorted by Priority.

• Edit – Click the blue item name to open a page where you can edit the item.

 

 

 

Adding a Coverage Symbol Group

1. Click Add.

2. In the Add Coverage Symbol Group dialog box, fill in the following required field:

 

Field

Description

Name

Name of the coverage symbol group. Choose a name that reflects the scope of coverages represented by the group.

3. Click OK to add the new coverage symbol group. The resulting Coverage Symbol Group home page enables you to edit your settings and add coverage symbol patterns to the group.

Deleting Coverage Symbol Groups

1. Navigate to the Coverage home page for each of the coverages that use the coverage symbol group, and then remove the coverage symbol group from the coverage. You cannot delete a coverage symbol group that is used by a coverage.

2. Navigate back to the home page for the coverage symbol group you were previously viewing.

3. Click Delete.

The deleted coverage symbol group is added to your active change list. It is not deleted from the PolicyCenter configuration until you commit your changes.
