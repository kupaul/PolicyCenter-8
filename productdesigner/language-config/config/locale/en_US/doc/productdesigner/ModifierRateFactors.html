Policy Line Modifier Rate Factors Page

Manage policy line modifier rate factors on the Rate Factors page. The Rate Factors page is available only if the modifier data type is Schedule Rate.

 

Column

Description

Sequence

Order in which the rate factor appears in PolicyCenter, relative to other rate factors.

Rate Factor Type

Rate factor type. Click to open the Policy Line Modifier Rate Factor home page where you can edit details of the rate factor type. Rate factor types are defined by configuring the PolicyCenter data model, as explained in the Product Model Guide.

Default Minimum

Default minimum value for this rate factor, if not overridden by a specified state minimum value. This value is also limited by the overall default minimum value specified in the Policy Line Modifier home page. Enter this value on the Policy Line Modifier Rate Factor home page.

Default Maximum

Default maximum value for this rate factor, if not overridden by a specified state maximum value. This value is also limited by the overall default maximum value specified in the Policy Line Modifier home page. Enter this value on the Policy Line Modifier Rate Factor home page.

Other operations

• Sort – Click any column header to sort by that column. Click again to reverse the sort order.

• Change order – If the table is sorted by Sequence, drag a single row to change the order in which it is listed. Alternatively, edit the sequence number to establish a new order. Sequence determines the order in which items appear in PolicyCenter PCF objects when they are sorted by Priority.

• Edit – Click the blue item name to open a page where you can edit the item.

 

 

 

Adding a Rate Factor

1. Click Add.

2. In the Add Rate Factor dialog box, fill in the following required field:

 

Field

Description

Rate Factor Type

Rate factor type. Select a rate factor type from those listed. Rate factor types are defined by configuring the PolicyCenter data model, as explained in the Product Model Guide.

3. Click OK to add the new rate factor. The resulting Policy Line Modifier Rate Factor home page enables you to edit your settings and define additional rate factor details.

Common operations

To revert changes to rate factors

 

The changes you make in the current page are stored in your active change list. You can

• Cancel all changes you have made in the current page by clicking Revert.

• Selectively cancel changes by clicking Changes to display the Changes panel, then clicking to open the Changes page, then clicking Revert in the Revert column.

Note:  When you click Revert, Product Designer reloads the entire product model from the workspace.

 

 

 

To delete rate factors

 

1. Select the row to delete by clicking anywhere in the row. To delete multiple items at once, select multiple rows.

2. Click Delete.

Deleted values are added to your active change list. They are not deleted from the PolicyCenter configuration until you commit your changes.

 

 

To navigate to related pages

Use the links under Go to to display pages where you can specify additional policy line modifier properties. The available links vary depending on the data type of the policy line modifier.
