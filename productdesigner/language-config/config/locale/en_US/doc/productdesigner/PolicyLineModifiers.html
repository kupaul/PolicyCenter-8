Policy Line Modifiers Page

Manage policy line modifiers on the Modifiers page. Both products and policy lines can have quote modifiers. This page defines modifiers that apply at the policy line level.

 

Column

Description

Sequence

Order in which the modifier appears in PolicyCenter, relative to other modifiers.

Name

Modifier name. Click to open the Policy Line Modifier home page where you can edit the details of the selected modifier.

Code

Unique identifier for the modifier. Established when you add a new modifier and thereafter cannot be changed. Must begin with a letter and be composed only of letters, numbers, and underscores.

Other operations

• Sort – Click any column header to sort by that column. Click again to reverse the sort order.

• Change order – If the table is sorted by Sequence, drag a single row to change the order in which it is listed. Alternatively, edit the sequence number to establish a new order. Sequence determines the order in which items appear in PolicyCenter PCF objects when they are sorted by Priority.

• Edit – Click the blue item name to open a page where you can edit the item.

 

 

 

Adding a Policy Line Modifier

1. Click Add.

2. In the Add Modifier dialog box, fill in the following required fields:

 

Field

Description

Code

Unique identifier for the modifier, which must begin with a letter and be composed only of letters, numbers, and underscores.

Name

Name of the modifier as it appears in PolicyCenter.

Modifier Type

Type of modifier selected from the modifier types defined in the data model for the line of business.

Data Type

Data type of the modifier: boolean, date, rate (decimal), or typekey.

Schedule Rate

Appears only if modifier Data Type is rate.

• Selected – Schedule rate modifier. PolicyCenter displays the Rate Factors tab on the Modifiers screen.

• Cleared – Experience modifier. PolicyCenter does not display the Rate Factors tab on the Modifiers screen.

Typelist

Appears only if you set Data Type to typekey. Select the applicable typelist from this drop-down list. The typelist contains a set of values from which a PolicyCenter user can select, such as a list of antitheft devices.

3. Click OK to add the new modifier. The resulting Policy Line Modifier home page enables you to edit your settings and define additional modifier details.

Common Operations

To revert changes to policy line modifiers

 

The changes you make in the current page are stored in your active change list. You can

• Cancel all changes you have made in the current page by clicking Revert.

• Selectively cancel changes by clicking Changes to display the Changes panel, then clicking to open the Changes page, then clicking Revert in the Revert column.

Note:  When you click Revert, Product Designer reloads the entire product model from the workspace.

 

 

 

To delete policy line modifiers

 

1. Select the row to delete by clicking anywhere in the row. To delete multiple items at once, select multiple rows.

2. Click Delete.

Deleted values are added to your active change list. They are not deleted from the PolicyCenter configuration until you commit your changes.

 

 

To navigate to related pages

Use the links under Go to to display pages where you can specify additional policy line properties. The available links vary depending on the data model of the line of business.
